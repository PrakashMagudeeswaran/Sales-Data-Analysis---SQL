Project Title: Retail Sales Analysis
This project aims to showcase fundamental SQL skills commonly applied by data analysts to examine, clean, and interpret retail sales data. It includes creating a retail sales database, conducting exploratory data analysis, and solving business-related problems using SQL queries. The project is well suited for beginners who want to develop a strong foundation in SQL and data analytics concepts.

Objectives
Database Setup: Design and populate a retail sales database using the provided sales dataset.

Data Cleaning: Detect and eliminate records containing missing or null values to improve data quality.

Exploratory Data Analysis: Perform initial analysis to understand data structure, trends, and key characteristics.

Business Analysis: Apply SQL queries to address business questions and generate meaningful insights from retail sales data.
